# EscapeRoom

<img width="2558" height="1242" alt="kép" src="https://github.com/user-attachments/assets/aae8c7ca-8269-4ac4-ad38-3bd0e68b1f5d" />

A first-person atmospheric puzzle tech demo built in Unity. The game features immersive mechanical puzzles and player sanity mechanics.

## Overview

You find yourself trapped in an unsettling environment with a singular goal: escape. As you explore and solve environmental puzzles, your character's sanity begins to fracture, altering your vision and perception of reality.

## Installation

Since a standalone executable is not yet compiled, you will need Unity to open and play the project.
